Slick Slider with Images
========================

You can slide images with "Slick Slider with Images".

If you like to slide different content elements than images, please use the
:ref:`Slick Slider with Content Element <editors_slickcontent>`.


Recommended
-----------

Number of columns should set to 1. Else you can get unwanted results.


TYPO3 6.2/7.6
-------------

* "Slick Slider with Images" was called "Slick Gallery".

* "Slick Slider with Images" isn't supported. Please use the :ref:`Slick Slider with
  Content Element <editors_slickcontent>` and add images.

* Properties are controlled by the Constant Editor. You need administration access. See
  :ref:`Integrators > Setup > TYPO3 6.2 / 7.6 > Slick Slider with Images <integrators-setup-76-slickimage>`

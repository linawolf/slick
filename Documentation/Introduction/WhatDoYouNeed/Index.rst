What do you need?
=================

- A TYPO3 installation only. The installation can be empty.


TYPO3 Versions
--------------

Slick should run proper with

- TYPO3 9.5 all features with Fluid Styled Content (fluid_styled_content)

- TYPO3 8.7 all features with Fluid Styled Content (fluid_styled_content)

- TYPO3 8.7 (limited features) with CSS Styled Content (css_styled_content)

- TYPO3 7.6 (limited features) with CSS Styled Content (css_styled_content)

- TYPO3 6.2 (limited features) with CSS Styled Content (css_styled_content)

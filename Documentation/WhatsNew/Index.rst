What's new
==========

Additional classes
------------------

5.1

You can assign a single CSS class to the slider.


Dots and arrows
---------------

5.1

Users can set up the offsets for the navigation dots and the next and previous arrows. 
Offsets are set by the plugin (tab layout) and not longer by the constant editor.


frame-before-class, frame-after-class
-------------------------------------

5.1

Respects frame-before-class and frame-after-class from tt_content


Hidden loading
--------------

5.1

The slider is hidden while the website is loading.
When JavaScript is loaded, the slider will appear.
